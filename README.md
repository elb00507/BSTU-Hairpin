# BSTU-Hairpin

# Особенности сайта

## Описание проекта

Этот сайт предлагает пользователям удобный интерфейс для взаимодействия с товарами и управления своим профилем. Он включает такие функциональные возможности, как регистрация, авторизация, редактирование профиля, переключение типов товара, подробное описание товаров и динамический дизайн.

---

## Функциональные возможности

### 1. Регистрация

- Пользователи могут создать учетную запись, предоставив необходимые данные (например, имя, email, пароль).

### 2. Авторизация

- Возможность войти в систему с использованием email и пароля.

### 3. Редактирование профиля

- Пользователи могут обновлять свои данные: имя, аватар, контактную информацию и другие настройки.

### 4. Переключение типов товара

- Удобная навигация между различными категориями товаров.
- Динамическое обновление списка товаров без перезагрузки страницы.

### 5. Описание товара

- Подробная информация о каждом товаре: название, цена, описание, характеристики, отзывы.
- Возможность добавления товара в корзину или избранное.

### 6. Динамический дизайн

- Адаптивный дизайн, обеспечивающий корректное отображение сайта на всех устройствах (компьютеры, планшеты, смартфоны).
